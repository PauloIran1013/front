Algoritmo: Verificar se um número é par ou ímpar

  1. Inicie o processo
  2. Solicite ao usuário que insira um número. Para isso, utilize o comando prompt para capturar a entrada do usuário como uma string.
  3. Converta a entrada do usuário para um número inteiro. Use a função parseInt() para garantir que o valor seja tratado como número.
  4. Verifique se o número é divisível por 2. Utilize o operador módulo (%) para verificar o resto da divisão do número por 2.
    Se o resto da divisão for 0, o número é par.
    Caso contrário, o número é ímpar.
  5. Exiba o resultado. Utilize o alert() ou console.log() para mostrar ao usuário se o número é "Par" ou "Ímpar". Finalize o processo.
